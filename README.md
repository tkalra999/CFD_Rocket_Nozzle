This project investigates compressible flow through a 2D axisymmetric converging–diverging rocket nozzle using ANSYS Fluent. The objective was to study supersonic flow behavior, Compressible Expansion Behaviour, and the effect of varying outlet pressure conditions on nozzle performance.
The simulation focused on:
•	Compressible flow Visualization
•	Compressible flow modelling
•	Structured mesh generation
•	Pressure ratio effects
•	Flow parameter extraction
